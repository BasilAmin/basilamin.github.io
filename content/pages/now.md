# Right now....
Since end of June, I've been at DogPatch Labs in Dublin city, attending the Patch Summer Accelerator Program. Myself and 3 others at the moment are working on Relay, a gantry based robotic chess system, utilizing the Lichess API to read and write to both a vitual and real-life chess games using an electromagnet mounted on a gantry, we disassembled from a Creality Ender 3 Pro.

In additon to that, I'm currently in the research stage of developing a centrally computed coordinated drone swarm system. To begin, it will be general purpose before I go niche in some area (probably industry use with data integration software too.)
